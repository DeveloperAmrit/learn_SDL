# Project Setup Guide

## Step 1: Download and Install Required Tools

### 1.1 Download MinGW64 (64-bit)
Follow the video tutorial to install MinGW64:
- [Download MinGW64](https://youtu.be/qH-9UAsI9h8?si=exETwfP0_ZF_HKZa)

### 1.2 Download Visual Studio Community Version
Download Visual Studio Community version from the official website:
- [Download Visual Studio](https://visualstudio.microsoft.com/downloads/)

### 1.3 Install Necessary Workloads in Visual Studio
Open the Visual Studio Installer and make sure to install the following workloads:
- **.NET desktop development**
- **Desktop development with C++**

Step 3: Connecting SDL and Visual Studio

Right click in source code folder to create a new main.c file
Now click on project in topmost toolbar > projectname properties
Warning : It sould be "projectname properties" and not only "properties". Otherwise you have not yet created main.C

Ensure "all configurations" in configuration
Click on C/C++ > General
Warning: This option will not appear if you have not created main.c
Add these to Additonal include directories one by one
C:\SDL2_libraries_\SDL2-devel-2.30.8-VC\SDL2-2.30.8\include
C:\SDL2_libraries_\SDL2_image-devel-2.8.2-VC\SDL2_image-2.8.2\include
C:\SDL2_libraries_\SDL2_mixer-devel-2.8.0-VC\SDL2_mixer-2.8.0\include

Now click Ok > apply

Now go to Linker > General

Add these to Additional library directories one by one
C:\SDL2_libraries_\SDL2-devel-2.30.8-VC\SDL2-2.30.8\lib\x64
C:\SDL2_libraries_\SDL2_image-devel-2.8.2-VC\SDL2_image-2.8.2\lib\x64
C:\SDL2_libraries_\SDL2_mixer-devel-2.8.0-VC\SDL2_mixer-2.8.0\lib\x64

Click Ok > Apply

Now go to Linker > Input

Add these to Additional dependencies
SDL2.lib; SDL2main.lib; SDL2_image.lib; SDL2_mixer.lib;

Click Ok > Ok
